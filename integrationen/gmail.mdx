Die Integration von Google Mail (als Teil von 'Google Workspace') ermöglicht den direkten Zugriff auf eingehende und ausgehende E-Mails, sodass unsere Agents Konversationen analysieren, automatisierte Antworten generieren und kontextbasierte Aktionen wie Follow-ups oder Erinnerungen initiieren können.
<Info>Solltest du dein Tool nicht unter den verfügbaren Integrationen finden, sende uns <a href="mailto:support@pzlr.ai">**hier**</a> einen Feature Request.</Info>

## Verfügbare Funktionen

Folgende Operationen sind mit unserer Hubspot-Integration grundsätzlich möglich und je nach Agent teilweise schon integriert.

<AccordionGroup>

  <Accordion title="Entwurf">
    - Entwurf erstellen/aktualisieren
    - Entwurf löschen
    - Entwürfe abrufen
  </Accordion>

  <Accordion title="E-Mail">
    - E-Mail senden/beantworten
    - E-Mail löschen
    - E-Mail abrufen
    - Als gelesen/ungelesen markieren
    - Label hinzufügen/entfernen
  </Accordion>

  <Accordion title="Unterhaltung">
    - Unterhaltung abrufen
    - Unterhaltung löschen
    - Gelöschte Unterhaltung wiederherstellen
    - Label hinzufügen/entfernen
  </Accordion>

  <Accordion title="Label">
    - Label erstellen/aktualisieren
    - Label löschen
    - Label abrufen
  </Accordion>

</AccordionGroup>

Per Custom API Calls können wir unseren Agents weitere, individualisierte Aufgaben zuweisen. Schicke uns hierzu gerne einen <a href="mailto:support@pzlr.ai">Feature Request</a>.

## Authentifizierung

Voraussetzung für die Nutzung der Gmail-API und die Integration in einen unserer Agents ist ein [Google Cloud](https://cloud.google.com/?hl=de) -Account.

Folgende Authentifizierungsmöglichkeiten werden von uns unterstützt:

- App Token
- Developer API-Key
- OAuth 2

Beim CRM-Booster (RAG-Agent) bzw. dem Sales-Agent kannst du über das Dashboard deine API Credentials verschlüsselt übermitteln.

<Note>Mehr Informationen zur Gmail-API findest du im offiziellen [Gmail API-Guide](https://developers.google.com/workspace/gmail/api/guides?hl=de).</Note>
