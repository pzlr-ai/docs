HubSpot ist ein vielseitiges CRM-Tool, das in mehrere Agents nahtlos integriert werden kann, um Vertriebs- oder Marketing-Prozesse effizient zu steuern, zu automatisieren und kontextbasiert mit Daten aus verschiedenen Quellen anzureichern.
<Info>Solltest du dein Tool nicht unter den verfügbaren Integrationen finden, sende uns <a href="mailto:support@pzlr.ai">**hier**</a> einen Feature Request.</Info>

**Warum das wichtig ist:**<br />
Durch die direkte Anbindung an HubSpot wird aus einem rein reaktiven CRM ein proaktiver Bestandteil deiner Workflows – ideal zur Reduktion manueller Aufwände, schnelleren Reaktionszeiten und einer nahtlosen User Experience für Kund:innen und Kandidat:innen.



## Verfügbare Funktionen

Hier sind sämtliche Operationen aufgeführt, die mit der Hubspot-Integration und unseren Agents möglich sind.

<AccordionGroup>

  <Accordion title="Kontakte">
    - Kontakt erstellen/aktualisieren
    - Kontakt löschen
    - Kontaktinformationen abrufen
    - Kontakt suchen
  </Accordion>

  <Accordion title="Unternehmen/Firma">
    - Firma erstellen/aktualisieren
    - Firma löschen
    - Firmeninformationen abrufen
    - Kürzlich erstellte/aktualisierte Firma abrufen
    - Firma suchen
  </Accordion>

  <Accordion title="Deals">
    - Deal erstellen/aktualisieren
    - Deal löschen
    - Dealinformationen abrufen
    - Kürzlich erstellte/aktualisierte Deals abrufen
    - Deal suchen
  </Accordion>

  <Accordion title="Aktivitäten">
    - Aktivität erstellen/löschen
    - Aktivität abrufen
  </Accordion>

  <Accordion title="Formular">
    - Alle Formularfelder abrufen
    - Dateien an Formular senden
  </Accordion>

  <Accordion title="Tickets">
    - Ticket erstellen/aktualisieren
    - Ticket löschen
    - Ticketinformationen abrufen
    - Ticket suchen
  </Accordion>

</AccordionGroup>

Über Custom API Calls können wir unseren Agents weitere, individualisierte Aufgaben zuweisen. Schicke uns hierzu gerne einen <a href="mailto:support@pzlr.ai">Feature Request</a>.









### Verfügbare Funktionen

**Lesender Zugriff**<br />
Abfrage von Kontakten, Deals, Pipelines, Aktivitäten etc.

**Schreibender Zugriff**<br />
Anlegen, Aktualisieren von Kontakten, Deals, Tasks

**Automatisierungen**<br />
Integration in Outreach-Sequenzen, Follow-ups

**Trigger**<br />
Aktionen auslösen bei Kontaktänderung, Deal-Stufenwechsel u.v.m.
