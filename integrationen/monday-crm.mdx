Monday CRM ist eine flexible und visuell strukturierte CRM-Lösung, die sich nahtlos in verschiedene Agents integrieren lässt, um Vertriebsprozesse, Aufgabenmanagement und Kundeninteraktionen zentral zu organisieren, zu automatisieren und dynamisch mit externen Datenquellen anzureichern.
<Info>Solltest du dein Tool nicht unter den verfügbaren Integrationen finden, sende uns <a href="mailto:support@pzlr.ai">**hier**</a> einen Feature Request.</Info>

## Verfügbare Funktionen

Folgende Operationen sind mit unserer Monday CRM-Integration grundsätzlich möglich und je nach Agent teilweise schon integriert.

<AccordionGroup>

  <Accordion title="Board">
    - Board erstellen
    - Board archivieren
    - Board abrufen
  </Accordion>

  <Accordion title="Board Eintrag">
    - Board Eintrag erstellen/aktualisieren
    - Board Eintrag löschen
    - Board Eintrag abrufen/filtern nach Reihenwert
    - Reihenwert für Board Eintrag ändern
  </Accordion>

</AccordionGroup>

Per Custom API Calls können wir unseren Agents weitere, individualisierte Aufgaben zuweisen. Schicke uns hierzu gerne einen <a href="mailto:support@pzlr.ai">Feature Request</a>.

## Authentifizierung

Folgende Authentifizierungsmöglichkeiten werden von uns unterstützt:

- API Token
- OAuth 2

Beim CRM-Booster (RAG-Agent) bzw. dem Sales-Agent kannst du über das Dashboard deine Monday Credentials verschlüsselt übermitteln.

<Note>Mehr Informationen zur Monday-API findest du in der offiziellen [API-Dokumentation von Monday.com](https://developer.monday.com/api-reference/docs/authentication).</Note>
