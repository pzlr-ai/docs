In der Basis-Version stehen zwei Indexierungsmöglichkeiten zur Verfügung. Es können sowohl komplette Sitemaps als auch einzelne Seiten als URL eingetragen werden. Zusätzlich können dem Agent verschiedene Dateiformate als Wissensbasis zur Verfügung gestellt werden.

## Websites 

Es besteht die Möglichkeit, sowohl eine Sitemap im `xml`-Format als auch eine einfache URL indexieren zu lassen. Die Sitemap-URL muss über das Internet öffentlich abrufbar sein.

<CodeGroup>

```javascript XML-Format
https://www.deinedomain.de/sitemap.xml
https://www.deinedomain.de/product-sitemap.xml
```

```python URL
https://www.deinedomain.de/produkt
https://www.deinedomain.de/event
```

</CodeGroup>

### Sitemap

Der Rackagent ist in der Lage, die in der Zeit UL enthaltenen Seiten zu exportieren und einzeln in seine Wissensbasis einzufügen. Wird eine Seiten URL erneut hochgeladen, werden sämtliche doppelten Inhalte überschrieben.

Sobald die Indexierung der Seiten erfolgreich abgeschlossen wurde und der Agent auf das Wissen zugreifen kann, erhält man eine Bestätigung E-Mail und der Status im Dashboard ändert sich in Live.

Es wird empfohlen, zusätzlich zur Sidemap UL auch einen Beschreibungstext einzugeben, wodurch die einzelnen Zeiten später einer bestimmten Zeit Map zugeordnet und dadurch gefiltert werden können.
